# Quotes Application


## Author: Evelyne UMUHIRE

## PROJECT DESCRIPTION
Quotes is an application where users can create quotes and have those quotes voted on whether they are inspirational or not.

The application allows the user to:

* Create Quotes.

* Vote on whether a quote is terrible or inspirational.
